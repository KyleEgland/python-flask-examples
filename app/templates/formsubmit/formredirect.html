Forms - Redirect submit

This page represents a (more or less) traditional form submission approach. Upon receiving the GET request from the browser for this endpoint, Flask renders the page and presents it to the user for completion. Once the user is satisfied with the form contents and submits, the browser then conducts a POST with the form's data. The POST is sent to the same endpoint which does validation before giving the form back to the user or re-directing the user to another page.
